- `test_terminate_closes_and_clears`
- `test_terminate_no_transcript_is_safe`

**Root cause:** tests ใช้ `PtySession.__new__(PtySession)` เพื่อ bypass `__init__` แต่ไม่ได้ set `session._pid = None` ก่อน call `terminate()`. เมื่อ `terminate()` เรียก `_tree_kill(self._pid)`, PyQt6 raise `RuntimeError: super-class __init__() of type PtySession was never called`.

**ยืนยันว่า pre-existing:** รัน `git stash` แล้ว test ยังล้มเหมือนกัน (ก่อน #52/#53 changes)
