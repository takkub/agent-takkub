🎯 Decision: cockpit_theme.py is the single source of truth

Newest, most complete, pixel-verified 4 rounds, and already the reference the user signed off on for Settings. All other UI files migrate to it, not the reverse. Migration is mechanical + semantic-preserving: swap literal greys → GROUND_*/TEXT_*, solid zinc borders → BORDER_*, the primary accent (active/selected/focus/CTA) → gold, fonts → ensure_fonts_loaded(). Keep role/provider/state/meter colors as tokens, not as gold.
